## 1. 今回のトップ3（言えなかった表現）

### ① 固定費 / 変動費（fixed expenses / variable expenses）

- **言いたかったこと**: 固定費と変動費を明確に分けて、変動費だけをコントロールする
- **実際の発話**: "the key is separating the fixed torque or fluctuate torque"（torque = トルク、全く別物）
- **完成形フルセンテンス（2つ）**:
  1. `The key is to clearly separate fixed expenses from variable expenses, and focus only on controlling the variable ones.`
  2. `I draw a line between fixed costs and variable costs, because only variable costs are things I can actually manage day to day.`
- **一行解説**: "torque" は機械のトルク。家計管理は必ず `expenses` か `costs`。`fixed vs variable` はセットで丸暗記。

### ② 裁量支出 / 予算内でやりくりする（discretionary spending / stay within budget）

- **言いたかったこと**: 裁量収入（採用収入）というやり方で、決めた金額だけでやりくりする
- **実際の発話**: "uncontrolled and manicating technique" / ja_log「大量収入って英語でなんて言うの？」
- **完成形フルセンテンス（2つ）**:
  1. `I use a discretionary spending method — I decide how much I can spend each month, and I stay within that budget.`
  2. `The idea is simple: set aside a fixed amount for discretionary spending, and live within that limit no matter what.`
- **一行解説**: 「自由に使える金額」は `discretionary` + 「その枠内でやる」は `stay within budget` / `live within that limit`。この2語が出るだけで一気にプロっぽく聞こえる。

### ③ 今月余裕がある / 今月厳しい（I have some wiggle room / Money is tight）

- **言いたかったこと**: 余裕がある時は外食、厳しい時は節約しよう、と切り替える

- **完成形フルセンテンス（2つ）**:
  1. `When I have some wiggle room, I tell myself "Let's go out for dinner this weekend." When money is tight, I switch gears and cut back.`

## 2. 今回のフォーカス弱点（1つだけ）

**「家計・お金の基礎名詞が全部 `torque` `feed` `fee` に溶けて混線する」**

お金の話では、抽象名詞（expenses / costs / income / savings / budget / spending）と動詞（spend / save / track / cut back / stay within）が噛み合って初めて意味が通る。今回はその名詞側が全滅したため、動詞がいくら頑張っても聞き手に伝わらない。

| en_log の実際 | 正しい完成形（2つ） |
|-------------|------------------|
| "the fixed torque or fluctuate torque" | ① `the fixed expenses and the variable expenses` / ② `fixed costs versus variable costs` |
| "the fluctuation feed is like the fashion passion, playing with old friends, and eating dogs" | ① `Variable expenses include things like eating out, hanging out with friends, and shopping.` / ② `Variable costs are stuff like dining out, leisure, and impulse purchases — things you choose to spend on.` |
| "this month, the practitioner fee is really fast" | ① `This month, the variable spending is running out fast.` / ② `We've burned through our variable budget really quickly this month.` |

**覚えるルール**: お金の話では `fee` ではなく `expense` / `cost` / `spending`。食事の話では `feed` ではなく `eat out` / `dining out`。この2語を間違えると聞き手は完全に迷子になる。

## 3. 意見スピーチの弱点（1つだけ）

**「仕組みを説明するスピーチ」なのに、プロセス動詞（track / monitor / compare）が無く、流れが見えない。**

### 自分の根拠（ja_log より）

> 「これをあの、まあ、一周目、二周目、三周目と、今いくらこれを使っているか、あとどのぐらい余裕があるか」
> 「定期的に、固定費、特にサブスクリプションであったりとか、あの、無駄なものはないかってことも見直しすることも、対処しています」

本人は「追跡する」「見直す」という**プロセスの動詞**を日本語ではサラッと使えているのに、英語ではこれらが全部 `calculate` / `find out` に圧縮されていて、ステップ感が消えている。結果、聞き手には「どうやって管理してるのか」の具体イメージが湧かない。

### 現状の発話（en_log より）

> "And every single week, I can... find out the gap between the boat and this mountain's face."

「毎週ギャップを確認する」と言いたかったはずが、`find out the gap` + 意味不明な比喩（"the boat and this mountain's face"）で崩壊している。`track`（追跡する）、`check` + against（〜と照らし合わせる）、`compare`（比較する）のどれも出てこない。

### 理想の言い方（2つ）

1. `Every week, I track how much I've already spent and compare it to my weekly budget. That way, I can see exactly how much wiggle room I have left.`
2. `At the end of each week, I check my spending against my plan. If I'm under budget, I treat myself. If I'm over, I cut back the next week.`

**覚えるルール**: 仕組み説明では **track / compare / check against / review** を動詞の主役に据える。`calculate` と `find out` だけでは「数字を出す」ニュアンスで止まり、「回している」感が伝わらない。
