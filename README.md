# English

英語スピーチ練習の記録と、Claude Code による弱点分析・復習サイクルのためのリポジトリ。

## これは何？

毎日 2 分間の即興英語スピーチを録音し、振り返りログを蓄積している。
目的は「練習して終わり」ではなく、**自分ができていないこと・苦手なことを定期的に可視化して、復習に繋げる**こと。

## 仕組み

1. スピーチ練習アプリで 2 分間即興で話す
2. 英語の書き起こし（`en_log.md`）と日本語の振り返り（`ja_log.md`）を記録
3. Claude Code がログを分析し、弱点パターンを抽出する

## 分析でできること

- **言えなかった表現の抽出**: Anki に入れられる単位で「日本語の意図 → 英語表現 → 出てこなかった理由」を整理
- **詰まりパターンの傾向分析**: 繰り返し詰まる構文・言い回しを特定
- **重点復習リスト**: 次の期間で集中的に回すべき表現タイプを提案

## データ形式

1日1フォルダ（`YYYY-MM-DD/`）:

```
english/
├── README.md
├── CLAUDE.md        # 分析ルール・運用仕様（Claude Code 用）
└── YYYY-MM-DD/
    ├── en_log.md    # 英語スピーチの書き起こし
    └── ja_log.md    # 日本語での振り返り（音声入力）
```

## 注意

public リポジトリのため、個人情報を含むログは push 前に確認しています。
